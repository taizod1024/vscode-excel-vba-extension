# Excel VBA module save/load

Excel VBA module save/load with vscode

![excel-vba](https://github.com/taizod1024/vscode-excel-vba-extension/blob/main/images/excel-vba.png?raw=true)

## Features

- **Load VBA from Excel Book** - Export VBA modules from an Excel workbook to the workspace
- **Save VBA to Excel Book** - Import modified VBA modules back into an Excel workbook
- **Compare VBA with Excel Book** - Compare workspace modules with the current Excel workbook
- **Open Excel Book** - Quick open Excel workbooks directly from the file explorer

## File Encoding

VBA module files (`.bas`, `.cls`, `.frm`) are automatically opened with Shift_JIS encoding to properly handle Japanese and other multi-byte characters commonly used in Excel VBA.
